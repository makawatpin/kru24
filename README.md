# ระบบบันทึกการออมเงินของนักเรียน

เว็บแอปสำหรับคุณครูใช้บันทึกเงินออมของนักเรียนรายวัน และส่งออกเป็นไฟล์ Excel ตามแบบฟอร์มราชการ (มีโลโก้ สพฐ. สี เส้นขอบ ฟอนต์ Angsana ครบ)

## คุณสมบัติ

- **บันทึกรายวัน** — เลือกเดือน/วัน แล้วกดปุ่ม 5/10/20 บาท ต่อนักเรียนแต่ละคน หรือพิมพ์จำนวนเอง มีปุ่มลัดทั้งห้องและทำเครื่องหมายวันหยุด
- **ตาราง** — มุมมองทั้งเดือนเหมือนแบบฟอร์ม (นักเรียน × วันที่ 1–31 + ยอดยกมา + รวม)
- **สรุป** — ยอดออมรวมทั้งปีของแต่ละคน แยกตามเดือน + ยอดรวมทั้งห้อง
- **ตั้งค่า** — แก้ข้อมูลโรงเรียน, เพิ่ม/ลบ/จัดลำดับรายชื่อนักเรียน, **นำเข้ารายชื่อจากไฟล์ Excel**
- **ส่งออก Excel** — เติมข้อมูลลงแบบฟอร์มต้นฉบับ (11 เดือน พ.ค.–มี.ค. + หน้าข้อมูล) หน้าตาเหมือนต้นฉบับ 100%
- รองรับมือถือ / แท็บเล็ต / คอมพิวเตอร์ — ข้อมูลบันทึกในเครื่องอัตโนมัติ

## ไฟล์ในโปรเจกต์

| ไฟล์ | หน้าที่ |
|------|--------|
| `ระบบบันทึกการออมเงิน.dc.html` | แอปหลัก (เปิดไฟล์นี้ในเบราว์เซอร์เพื่อใช้งาน) |
| `support.js` | รันไทม์ของ Design Component |
| `xlsx-export.js` | ตัวสร้างไฟล์ Excel (สำรอง) |
| `xlsx-template.js` | ตัวเติมข้อมูลลงแบบฟอร์มต้นฉบับ (ใช้หลัก) |
| `savings-template-data.js` | แบบฟอร์ม Excel ต้นฉบับ (base64) |
| `mockup.html` | ภาพตัวอย่างหน้าจอมือถือ |
| `assets/savings-template.xlsx` | ไฟล์แบบฟอร์มต้นฉบับ |

## วิธีใช้งาน

เปิด `ระบบบันทึกการออมเงิน.dc.html` ในเว็บเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร

> หมายเหตุ: ข้อมูลถูกบันทึกใน localStorage ของเบราว์เซอร์เครื่องนั้น หากต้องการเก็บถาวร ให้ส่งออกเป็น Excel เป็นระยะ
